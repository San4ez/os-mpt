Лабораторная #5 по ОС

Лабораторная работа №5 - Установка ОС на VirtualBox.

Цель работы: Получить практические навыки работы с виртуальными машинами.

Установить дистрибутив ОС можно несколькими способами

  1. Установить как основную ОС
  2. Установить паралельно основной ОС
  3. Установить на загрузочный диск или флешку (Live-USB/CD)
  4. Установить на виртуальную машину

Для нас самый удобный будет последний вариант, в качестве среды виртуализации будем использовать VirtualBox. он бесплатен и скачать его можно на оф.сайте под любую платформу.

В качестве другой ОС будем использовать Ubuntu, скачать iso образ можно на оф.сайте, советую 16-ую версию.
